# 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/mostafa_toolkit.git
```

Go to the project

```bash
cd mostafa_toolkit
```

Install

```bash
pip install -e .
```

---

# ⚡ Quick Start

```python
from mostafa_toolkit import *

# Load dataset
df = pd.read_csv("house_prices.csv")

# EDA
statistical_analysis(df)

# Visualization
plot_correlation(df)

# Create model
model = RandomForestRegressor()
```
